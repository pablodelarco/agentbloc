# Plan 15-01: Anticipation Heuristics + Arco Rooms Anticipated Fixture

**Wave:** 1
**Depends on:** none (parallelizable across the 3 tasks)
**Closes:** ANTIC-02, ANTIC-04, ANTIC-05 (3 of 5 Phase 15 requirements)

## Goal

Ship the consulting-product credibility foundation: an evidence-backed heuristics map that drives Designer's anticipation pass, the Arco Rooms 5-agent fixture that Phase 16 uses as the canonical E2E test case, and the declined.json schema demonstration.

## Wave Sequencing

The 3 tasks are independent and parallelizable. Recommended order is heuristics first (because the fixture cites it), then fixture, then declined.json (smallest). Atomic commits: 1 per task.

## Task Sketches

### Task 1: anticipation-heuristics.md

5 H2 sections, one per business type. Each section is 4 blocks:

```markdown
## Business type: <type-id>

Brief 2-sentence description of what this business shape looks like.

### Anticipated agents

| Agent ID | Role | 1-line rationale |
|---|---|---|
| ... | ... | ... |

### Evidence sources

1. [URL](URL) , <last-checked YYYY-MM-DD> , 1-line summary of what this source establishes
2. [URL](URL) , <last-checked YYYY-MM-DD> , 1-line summary
3. [URL](URL) , <last-checked YYYY-MM-DD> , 1-line summary

### When NOT to anticipate

Paragraph documenting business sub-shapes where these anticipated agents are NOT appropriate (e.g., single-property landlord with 1 tenant doesn't need a profitability analyst).
```

5 mappings:
1. **rental-property-management** -> Profitability Analyst + Incident Tracker
2. **ecommerce** -> Returns Analyst + Inventory Forecaster
3. **freelance-services** -> Cashflow Forecaster + Lead Pipeline Tracker
4. **restaurant** -> Inventory Reconciler + Reputation Monitor
5. **professional-services** -> Utilization Tracker + Renewal Anticipator

Each mapping cites 3+ independent reputable sources (industry reports, regulatory documents, framework docs, vendor case studies, peer-reviewed publications). Sources MUST be triangulated (not three blog posts citing each other).

Top of file: Table of Contents + brief introduction explaining the consulting-product positioning + "How to add a new mapping" footer pointing future contributors at the 4-block schema.

- Body Section `## Failure Details`: concrete evidence (file, field, step, exact error).
- Body Section `## Partial State`: list of files that WERE written before halt; may need manual rollback.
- Body Section `## Resumption Advice`: what user should fix before re-running the deploy.

halt_reason enum (D-70, 6 values):
1. `template-load-failure` , missing/corrupt template file, or template contains `{%` directive token

5. `verification-failed` , Check 1 (agents list) hard-fail, or Check 2 hard-fail on required integration
6. `user-rejected-diff` , user rejected the D-61 diff approval prompt

</halt_protocol>
